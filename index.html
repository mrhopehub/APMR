Introduction

Arduino Power Meter Reader (APMR) is an open source hardware and software project. APMR reads the power (W) and energy (Wh) readings of RS485/Modbus communicating power meters at a constant interval. Readings are recorded on an SD card and/or sent over the Internet using a HTTP RESTful web service.

Version 2.5 is now released. See the Release History section to review what has changed. Since Version 2.0 was released you can now modify settings and view files from your browser. Here are the URLs:

http://<ip.address>/         - view the current readings.
http://<ip.address>/settings - view/modify settings.
http://<ip.address>/unsent   - view the unsent readings file.
http://<ip.address>/files    - view SD card info and files.

The Hardware

APMR used an Arduino Mega 2560 with two custom shields: the RTC (real-time clock) Shield and the RS485 Shield.

Each custom shield can be create using a Arduino Proto Shield and the following schematics:

  • The RTC Shield schematic (based on the SparkFun Real Time Clock Module)
  • The RS485 Shield schematic

The In-Home Display

In Version 2.5, we have added an in-home display (IHD) prototype that works with APMR. The IHD uses three colour LEDs (green, yellow, and red) as ambient indicators. These colours signal how much energy you are currently consuming relative to your past consumption.
