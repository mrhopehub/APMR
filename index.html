Introduction

Arduino Power Meter Reader (APMR) is an open source hardware and software project. APMR reads the power (W) and energy (Wh) readings of RS485/Modbus communicating power meters at a constant interval. Readings are recorded on an SD cards and/or sent over the Internet using a HTTP RESTful web service.

We are currently working on Version 2.0 which will include some great features (listed in the Release History section).

Stay tuned.

The Hardware

APMR used an Arduino Mega 2560 with two custom shields: the RTC (real-time clock) Shield and the RS485 Shield.

Each custom shield can be create using a Arduino Proto Shield and the following schematics:

  • The RTC Shield schematic Based on the SparkFun Real Time Clock Module.
  • The RS485 Shield schematic
